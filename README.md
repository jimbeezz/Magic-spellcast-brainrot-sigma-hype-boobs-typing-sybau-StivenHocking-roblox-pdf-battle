# SpellCaster: Arena

Игра, которая прокачивает скорость и точность печати через магические бои в реальном времени.

## Концепция

Игрок управляет магом и кастует заклинания через ввод сложных строк.  
Чем мощнее заклинание, тем сложнее комбинация: регистр, цифры, спецсимволы, смешение кириллицы и латиницы.

Примеры формата ввода:

- `Fb8alL` (базовая техничная комбинация)
- `пр@ок^az/*za` (смешанная сложная комбинация)
- `☻↓→♫kZ` (редкие и alt-символы)

## Боевая система

- Бой идет в реальном времени по таймерам.
- Враг готовит атаку, игрок должен успеть ввести комбинацию до истечения окна каста.
- Ошибка во вводе отменяет заклинание и создает уязвимость по темпу.
- У части заклинаний есть эффекты: замедление, прерывание, пробитие щита, контроль.
- В бою всегда доступен UI-список выученных заклинаний с паттернами, чтобы игрок не учил их наизусть.
- Перед боем игрок выбирает loadout (ограниченный набор заклинаний на матч).

## Прогрессия

- За победы начисляется опыт.
- Опыт тратится на:
  - открытие новых заклинаний;
  - увеличение окна ввода (скорость каста);
  - рост маны и здоровья;
  - освоение более сложных паттернов ввода.

## Режимы игры

### Arena (основной)

Последовательные бои против врагов с уникальными уязвимостями, сопротивлениями и механиками.

### Exam (дополнительный)

Режим тренировки и проверки навыка:

- игрок кастует цепочку заклинаний без ошибок;
- задача: набрать максимум успешных кастов за ограниченное время;
- любой промах ломает серию (или завершает попытку в strict-варианте);
- лидерборд по очкам, серии и скорости.

## Враги и боссы

В проекте запланированы:

- базовые враги стихий (агрессор, танк, стеклянная пушка);
- тактический босс с адаптивными резистами;
- хаотичный босс, искажающий правила ввода;
- финальный босс, требующий высокий уровень mastery-паттернов.

## Дизайн-принципы

- Высокий skill ceiling, но мягкий onboarding.
- Сложность должна ощущаться честной, а не случайной.
- Ввод и фидбек обязаны быть мгновенными и читаемыми.
- Каждая новая механика должна менять тактику, а не только увеличивать HP врагов.
